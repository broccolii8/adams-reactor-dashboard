# ADAMS 차세대 원자로 인허가 대시보드 (발행)

이 저장소는 [ADAMS 인허가 인텔리전스 대시보드](https://github.com/) 프로젝트의 **발행 전용
스냅샷**입니다. 개발 코드는 여기 없습니다 — `dashboard.json`/`index.html`만 매일 자동 갱신되어
push됩니다.

- `dashboard.json`: 카카오톡 챗봇 등 자동화용 데이터
- `index.html`: 사람이 보는 요약 페이지 (GitHub Pages로 호스팅)

데이터 출처: NRC ADAMS Public Search API (공개 문서 기반).
